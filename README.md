# AI Task Manager

**AI Task Manager** е интелигентно уеб приложение за управление на задачи, разработено с **Django** и интегрирано с мощния модел **Gemini 2.5 Flash** на Google. Проектът не само ви позволява да следите ежедневните си ангажименти, но и ви предоставя персонализирани, мотивиращи съвети от изкуствен интелект (AI) въз основа на текущия ви списък от задачи.

---

## ✨ Функционалности

- **🔐 Потребителска автентикация:** Сигурна система за регистрация, вход и изход от профила. Всеки потребител вижда единствено своите задачи.
- **📅 Управление на задачи:** Създаване на задачи със заглавие, описание, категории, ниво на приоритет и крайни срокове.
- **📊 Динамична статистика:** Графична лента за прогрес (Progress Bar) и броячи в реално време, показващи общия брой задачи, чакащите и завършените.
- **⚠️ Автоматично отчитане на просрочия (Overdue):** Системата сравнява крайния срок на всяка незавършена задача с текущото време и оцветява закъснелите задачи в червен цвят.
- **🧠 Интелигентен AI Асистент:** Интеграция с `google-genai` (модел `gemini-2.5-flash`), който анализира вашите чакащи задачи и генерира кратък, мотивиращ съвет от максимум 2 изречения за това откъде да започнете деня си.

---

## 🛠️ Използвани технологии

- **Backend:** Python, Django Web Framework
- **Frontend:** HTML5, CSS3, Bootstrap 5 (за адаптивен и модерен дизайн)
- **AI Интеграция:** `google-genai` SDK (Google Gemini API)
- **База данни:** SQLite3 (по подразбиране за разработка)

## 🚀 Инсталация и Стартиране

Следвайте тези стъпки, за да стартирате проекта локално на вашия компютър:

### 1. Клониране на хранилището
```bash
git clone [https://github.com/VladislavDoychinov/ai-task-manager.git](https://github.com/VladislavDoychinov/ai-task-manager.git)
```
### 2. Отваряне на папката на проекта
```bash
cd ai-task-manager
```
### 3. Създаване и активиране на виртуална среда
```bash
python -m venv venv
```
Windows:
```bash
venv\Scripts\activate
```
Linux/macOS:
```bash
source venv/bin/activate
```
### 4. Инсталиране на зависимостите
```bash
pip install django google-genai python-dotenv
```
```markdown
⚠️ **Важно преди стартиране:** > Проектът използва относителни импорти между папките. За да избегнете грешката `ModuleNotFoundError`, е необходимо да укажете на Python да търси модули в коренната папка на проекта.
```
За Windows:
```bash
set PYTHONPATH=.
```
За Linux/macOS
```bash
export PYTHONPATH=.
```
### 4. Създаване на .env
```markdown
⚠️ **Важно преди стартиране:** > Създайте файл с име .env в папка src/ на проекта и добавете вашия API ключ от Google AI Studio:
- Забележка: Файлът .env се игнорира от Git от съображения за сигурност.
```
```bash
GEMINI_API_KEY=вашият_таен_gemini_api_ключ_тук
```
### 5. Изпълнение на миграциите към базата данни
```bash
python src/manage.py makemigrations
python src/manage.py migrate
```
### 6. Стартиране на локалния сървър
```bash
python src/manage.py runserver
```
Отворете на браузъра си адрес: http://127.0.0.1:8000/
